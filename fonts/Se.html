Селен

Селен – хрупкий, блестящий на изломе неметалл серого цвета.

Физические свойства

  • Плотность = 4,79 г/см³.
  • Агрегатные состояния. Твердеет при температуре 25 °С.
  • Растворимость в воде. Не растворяется.

История открытия

Открыт в 1817 г. Берцелиусом, который так рассказывает об этом открытии:
"Я исследовал в сотружестве с Готлибом Ганом метод, который применяют для производства серной кислоты в Грипсхольме. Мы обнаружили в серной кислоте осадок, частью красный, частью светло-коричневый."

Способы получения

Получение селена проходит в два этапа:

  1. SeO2 + H2O = H2SeO3
  2. H2SeO3 + 2SO2 + H2O = 2H2SO4 + Se

Нахождение в природе

Основные месторождения селена связаны с месторождениями серы и сульфидов. Вещество содержится в почвах, во многих растениях, в организме животных и человека.

Химические свойства

  1. Реакция горения:
    • Se + O2 = SeO2
  2. Селен окисляется азотной кислотой:
    • Se + 4HNO3 = SeO2 + 4NO2 + 2H2O
  3. С металлами:
    • Se + 2Na = Na2Se
  4. С галогенами (кроме иода):
    • Se + 3F = SeF6
    • Se + 2Cl = SeCl4
    • 2Se + Cl2 = Se2Cl2
    • Se + 2Br2 = SeBr4
    • 2Se + Br2 = Se2Br2
  5. Реагирует с щелочами:
    • 3Se + 6NaOH = Na2SeO3 + 2Na2Se + 3H2O
