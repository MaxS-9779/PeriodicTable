Кремний

Кремний — вещество тёмно-серого цвета с металлическим блеском. Кремний встречается только в виде сложных веществ, таких как горный хрусталь, кварц, кремнезем.

Физические свойства

  • Плотность = 2,33 г/см³.
  • Агрегатные состояния. Существует только в твёрдом.
  • Температура плавления 1414°С
  • Растворимость в воде. При растворении кремний обычно не ионизируется; он присутствует в виде ортокремниевой кислоты (H4SiO4 или Si(OH)4).

История открытия

В свободном виде впервые выделил (в виде аморфного порошка) в 1824 г. шведский химик Й. Я. Берцелиус; он же понял, что кремний — химический элемент.

Способы получения

Свободный кремний получается при прокаливании мелкого белого песка (диоксида кремния) с магнием:

SiO2 + 2 Mg = 2MgO + Si

Нахождение в природе

Образует множество минералов. В основном это алюмосиликаты. Кремний входит в состав кварца, горного хрусталя, аметиста, цитрина, авантюрина, яшмы и тигрового глаза.

Химические свойства

  1. Взаимодействие с галогенами (При нормальной температуре кремний активно реагирует только с газообразным фтором):
    • Si + 2F2 = SiF4
  2. Реакция с кислородом:
    • Si + O2 = SiО2
  3. Реакция с щелочами:
    • Si + 2NaOH + H2O = Na2SiO3 + 2H2
  4. Реакция с кислотами:
    • Si + 2HNO3 = SiO2 + NO + NO2 + H2O
