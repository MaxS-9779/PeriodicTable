Углерод

Углерод — чёрное порошкообразное или твёрдое вещество. В чистом виде не имеет запаха. Известно много разновидностей углерода, самые известные — алмаз и графит.

Физические свойства

  • Плотность, г/см³: алмаз 3,52, графит 2,27.
  • Агрегатные состояния. Существует только в твердом.
  • t плавления = 3550°С

История открытия

В 1791 году английский химик Теннант первым получил свободный углерод; он пропускал пары фосфора над прокалённым мелом, в результате чего образовывались фосфат кальция и углерод.

Способы получения

  • Карбонизация
  • Технический углерод получают с помощью способа термического разложения углеводородов в газовой камере.

    • Нахождение в природе

    Встречается в виде простых веществ — алмаза и графита, он входит в состав углекислого газа — СО2, карбонатов: доломита — СаMg(CO3)2, мрамора — CaCO3, природного газа, нефти, каменного угля.

    Химические свойства

      Углерод, как восстановитель:

    1. Взаимодействие с неметаллами: кислородом, фтором и серой
      • C + O2 = CO2 (кислород в избытке)
      • 2C + O2 = CO (кислород в недостатке)
      • C + 2F2 = CF4
      • C + 2S = CS2
    2. Взаимодействие с оксидами щелочных металлов:
      • 2ZnO + C = 2Zn + CO2
      • ZnO + C = Zn + CO (C в избытке)
    3. При взаимодействии с оксидами щёлочноземельных металлов и алюминия образуются карбиды:
      • CaO + 3C = CaC2 + CO
      • Al2O3 + 9C = Al4C3 + 6CO
    4. Взаимодействие с оксидами неметаллов:
      • C + CO = CO2
      • 2C + SiO2 = Si + 2CO
    5. Взаимодействие с концентрированными серной и азотной кислотами. При нагревании углерод окисляется концентрированными серной и азотной кислотами до оксида углерода(IV):
      • C + H2SO4 = CO2 + 2SO2 + 2H2O
      • C + 4HNO3 = CO2 + 4NO2 + 2H2O

      Углерод, как оксислитель:

    1. Взаимодействие с неметаллами: При высокой температуре углерод реагирует с металлами, образуя карбиды
      • 2C + Ca = CaC2
      • 3C + 4Al = Al4C3
    2. Взаимодействие с неметаллами:
      • C + H2 = CH4
      • C + Si = SiC
