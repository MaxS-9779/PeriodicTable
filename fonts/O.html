Кислород

Кислород — газ без цвета, вкуса и запаха (при н.у.).

Физические свойства

  • Плотность = 1,25 г/л
  • Агрегатные состояния. Жидкий кислород - жидкость бледно-синего цвета. Твёрдый кислород — синие кристаллы.
  • t плавления = -218,8°С.
  • Малорастворим в воде.

История открытия

Впервые кислород в свободном виде получили в конце XVIII веке почти одновременно несколько ученых из разных стран. В 1774 г. английский химик Джозеф Пристли (1733—1804) получил кислород нагреванием оксида ртути. Шведский химик К.В. Шееле получил кислород из селитры в 1771 г., но опубликовал свой эксперимент только в 1777 г.

Способы получения

  • В 1774 г.:
    2HgO = 2Hg + O2
  • В промышленности: из жидкого воздуха.
    2KMnO4 = K2MnO4 + MnO2 + O2
    2H2O2 = 2H2O + O2
    2KClO3 = 2KCl + 3O2
    2KNO3 = 2KNO2 + O2

    • Нахождение в природе

    Кислород — самый распространённый в земной коре элемент, на его долю приходится около 60 % массы земной коры. В составе воздуха: 20,94% по объёму. В морской и пресной воде содержание по массе — 85,82 %.

    Химические свойства

    1. Взаимодействие с металлами:
      • O2 + Na = Na2O2 (реагируя с натрием образует пероксид)
      • O2 + Al = Al2O3
      • O2 + Fe = Fe3O4 (FeO · Fe2O3
    2. Взаимодействие с неметаллами:
      • O2 + S = SO2
      • O2 + N2=2NO
